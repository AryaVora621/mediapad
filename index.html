the pcb

Designed in KiCad

This was my first real PCB. I did the schematic first, then placed everything and routed all the traces by hand. It's small enough to sit next to a keyboard without being in the way. The design is done and ready to fab.

🧠
Seeed XIAO RP2040
Tiny microcontroller with USB-C built in. Runs CircuitPython which makes the firmware super easy to write and change.
U1 — XIAO RP2040
🎛
Rotary Encoder
Turn it to change volume, click it to mute. EC11-compatible so it's easy to source and swap out.

firmware

Running KMK

KMK is keyboard firmware written in CircuitPython. The Mediapad shows up as a regular USB keyboard to your computer, no drivers needed. Changing keybindings is just editing a python file and saving, no reflashing.

code.py
# Mediapad KMK config from kmk.kmk_keyboard import KMKKeyboard from kmk.keys import KC from kmk.modules.encoder import EncoderHandler keyboard = KMKKeyboard() encoder_handler = EncoderHandler() # 3 keys: play/pause, prev, next keyboard.keymap = [[ KC.MPLY, KC.MPRV, KC.MNXT, ]] # encoder: vol up, vol down, mute on click encoder_handler.map = [[(KC.VOLU, KC.VOLD, KC.MUTE)]]
🔌
just plug it inshows up as a standard keyboard. works on mac, windows, and linux with zero setup.
🐍
CircuitPythonthe XIAO mounts as a USB drive. edit code.py and save. no compile step, no flashing tools.
🎛
encoder supportKMK handles rotary encoders natively. clockwise and counterclockwise each map to their own key.
🔁
easy to remapwant scroll or zoom instead of volume? just change two lines. no touching hardware.
parts list

What's in it

All the components for the build. Parts marked "in kit" are included in the Hackpad component kit from Hack Club.

PartDetails
Microcontroller in kit Seeed Studio XIAO RP2040, USB-C, 264KB SRAM, 2MB Flash
Rotary Encoder in kit EC11-compatible, 20 detents, built-in click switch
Switches in kit 3x MX-compatible mechanical switches
Keycaps in kit 1U MX-compatible keycaps
OLED in kit SSD1306 128x32 over I2C, footprint already on the PCB, adding it next
PCB 2-layer KiCad design, ordering from JLCPCB
Case 3D printed, designed in Onshape, v2 redesign in progress
Firmware KMK on CircuitPython, open source
